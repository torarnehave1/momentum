# Vegvisr Momentum Template

Starter template for Vegvisr apps (Cloudflare Pages + React + Tailwind v3).

## Included
- AuthBar + magic link flow
- EcosystemNav app bar
- Language selector + i18n scaffold
- Tailwind v3 configuration aligned with other Vegvisr apps

## Quick start
```bash
npm install
npm run dev
```

## Customize
- Replace `src/assets/app-icon.png` + `src/assets/app-logo.png`.
- Update translations in `src/lib/i18n.ts`.
- Replace the placeholder content in `src/App.tsx` with your app features.
